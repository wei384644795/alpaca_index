# 

##  本项目使用 了  vue.js  element-ui.js 
### 主要使用 技术 vue-cli  脚手架   前提是 node.js 
###  npm install  安装相关的 js 
```
npm install
```

### 开启服务  开启后可以 通过  http://localhost:8080/ 访问
```
npm run serve
```

###  将 项目 构建到 dist 目录中 
```
npm run build
```

### Customize configuration
See [Configuration Reference](https://cli.vuejs.org/config/).
